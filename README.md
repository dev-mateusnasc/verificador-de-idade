# Verificador de Idade

Aplicação web desenvolvida com **HTML, CSS e JavaScript** que calcula automaticamente a idade do usuário com base no ano de nascimento informado e exibe dinamicamente uma imagem correspondente à faixa etária e ao gênero selecionado.

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de **JavaScript, manipulação do DOM e interação com o usuário.**

---

## Demonstração

A aplicação permite que o usuário:

1. Informe o **ano de nascimento**
2. Selecione o **gênero**
3. Clique em **verificar**

Após isso, o sistema:

- calcula a idade automaticamente
- identifica a faixa etária
- exibe uma mensagem personalizada
- mostra uma imagem correspondente ao perfil detectado

---

## Funcionalidades

- Captura de dados através de **inputs HTML**
- Validação de dados inseridos pelo usuário
- Cálculo automático da idade utilizando **JavaScript Date()**
- Manipulação dinâmica de elementos utilizando **DOM**
- Alteração automática de imagens baseada em:
  - criança
  - jovem
  - adulto
  - idoso
- Interface simples e intuitiva

---

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
